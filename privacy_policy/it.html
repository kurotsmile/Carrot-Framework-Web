Log files
{{site_name}} segue una procedura standard di utilizzo dei file di registro. Questi file registrano i visitatori quando visitano siti web. Tutte le società di hosting fanno questo e una parte dell'analisi dei servizi di hosting. Le informazioni raccolte dai file di registro includere indirizzi IP (protocollo Internet), tipo di browser, provider di servizi Internet (ISP), data e ora, pagine di riferimento/uscita ed eventualmente il numero di clic. Questi non sono collegati ad alcuna informazione che lo sia personalmente identificabile. Lo scopo delle informazioni è l'analisi delle tendenze, l'amministrazione del sito, il monitoraggio movimento degli utenti sul sito Web e raccolta di informazioni demografiche.
